# GitHub 命盘

GitHub 命盘：把你的 GitHub 贡献、仓库与协作转为星宿命盘星图，支持导出、团队合盘，并提供成就、大运年轮与多风格渲染体验。

## 功能一览
- 单人起盘：Canvas/SVG/文字三种渲染；导出图片或复制文案；年轮星图、成就、守护灵、大运
- 双人合盘：两人相性解读
- 团队星图：2–8 人团队五行分布、相性矩阵

## 快速开始
```bash
npm install
npm run dev
```

生产构建：
```bash
npm run build
npm run start
```

## 部署说明
项目支持部署到 Vercel、Cloudflare Pages 等平台。

### 本地运行
1. 克隆项目：`git clone <仓库地址>`
2. 进入目录：`cd GitHubStarChart`
3. 安装依赖：`npm install`
4. 启动开发服务器：`npm run dev`
5. 访问：`http://localhost:3000`

### 生产部署
1. 构建项目：`npm run build`
2. 启动生产服务器：`npm run start`
3. 或部署到你选择的云平台
